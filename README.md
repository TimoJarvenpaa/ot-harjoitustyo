# Pong-klooni

Sovelluksen voi käynnistää komennolla
```
mvn compile exec:java -Dexec.mainClass=pong.ui.PongUI
```

Testit voidaan suorittaa komennolla
```
mvn test
```

Testikattavuusraportti voidaan luoda komennolla
```
mvn test jacoco:report
```
